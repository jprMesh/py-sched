Python Schedule Viewer
========

* schedule2.py is compatible with Python 2.6 and up.

* schedule3.py is compatible with Python 3.3.

Usage
--------
```
$ python schedule2.py
```
or
```
$ python3 schedule3.py
```

Legal colors
--------
* red
* green
* blue
* cyan
* yellow
* gray
* magenta
* black
* pink
* hot pink
* lavender
* light green
* spring green
* chartreuse
* medium blue
* purple
* etc.
